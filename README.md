# Holiday Omnichannel Retail Analytics (FP20 Challenge 33)
![Holiday Omichannel Banner]![Holiday Omichannel](https://github.com/user-attachments/assets/712d1266-9b7b-4005-a902-45ce51a641df)


## Overview
This project analyzes a holiday retail dataset to evaluate sales, profit, returns,
logistics efficiency, and customer omnichannel behavior using Power BI.

## Key Business Questions
- Which channels and campaigns drove the most profit?
- How do returns and discounts vary by category?
- Where do shipping delays occur?
- Do omnichannel customers generate higher value?

## Key Insights
- Web channel led performance with $26.7M revenue and $6.7M profit
- Campaign CM generated the highest profit
- 44.25% of customers were omnichannel and had larger basket sizes
- Footwear had the highest return rate; Grocery had the highest discounts
- Shipping delays were concentrated in Warehouse W03 (DHL) and W05 (UPS)

## Tools Used
- Power BI
- DAX
- Data Modeling
- Interactive Dashboards
